# Security Policy

If you discover a security vulnerability in this project, please report it privately so we can respond and patch before details are made public.

Preferred reporting methods:

1. Create a GitHub issue and prefix the title with `SECURITY:` and include sensitive details only if you are comfortable. Maintain confidentiality if necessary.
2. If you prefer a private report and the repository supports it, use GitHub Security Advisories.

We will acknowledge receipt within 72 hours and respond with next steps. Do not publicly disclose vulnerabilities until a fix is available.
